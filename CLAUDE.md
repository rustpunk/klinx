# CLAUDE.md

Claude Code should use the shared agent onboarding docs for this repository.

Read first:

1. [`AGENTS.md`](AGENTS.md)
2. [`docs/ai/00_READ_THIS_FIRST.md`](docs/ai/00_READ_THIS_FIRST.md)
3. The nearest local `AGENTS.md` for the directory being edited, if one exists.

The detailed durable documentation set lives in [`docs/ai/`](docs/ai/). Treat
`AGENTS.md` and `docs/ai` as the source of truth for agent rules, architecture,
commands, design invariants, testing expectations, performance notes, glossary
terms, and open questions.

Do not duplicate repository guidance here. If the shared guidance changes,
update `AGENTS.md`, the relevant local `AGENTS.md`, or `docs/ai` instead.
